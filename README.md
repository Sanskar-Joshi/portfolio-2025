# 👨‍💻 Sanskar Joshi - Personal Portfolio
Welcome to my personal portfolio repository! This project is a live showcase of my skills as a Web Developer, built entirely with core web technologies to ensure maximum performance and clean code.

### 🌟 Key Features
* **Fully Responsive:** Optimized layout for Mobile, Tablet, and Desktop screens.
* **Interactive UI:** Custom CSS hover effects, scroll animations, and a "Spotlight" cursor effect.
* **Serverless Backend:** A unique dual-submission contact form that sends emails via Web3Forms and logs data to Google Sheets simultaneously.
* **Single Page Application (SPA) Feel:** Smooth section transitions using JavaScript.

### 🛠 Teck Stack
* **Frontend:** HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+)
* **Tools:** VS Code, Google Apps Script, Boxicons
